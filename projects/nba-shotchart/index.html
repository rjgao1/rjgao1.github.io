NBA-Shotchart

An interactive dashboard for NBA player shot data.

Note: currently under maintaince as NBA has imposed CORS restrictions that prevent the package from being used in a browser directly

Work Summary

  • Built and deployed a dashboard using React JS, D3 and Ant Design to visualize NBA players’ shot performance

  • Improved user experience by adding 4 data filters and an autocomplete search bar providing player image preview

  • Developed 2 shot themes to demonstrate frequency-efficiency and distance-action relations

  • Technologies involved: JavaScript, React, Google App Engine, NBA API, D3, Ant Design

Built with React, d3-shotchart (https://www.npmjs.com/package/d3-shotchart ), a Node.js client for stats.nba.com API endpoints (https://www.npmjs.com/package/nba) and Ant Design (https://ant.design).

The hexbin chart visualizes the frequency and efficiency of shots made on the court.
In the hexbin mode, user can filter shots by minimum frequency using the slide bar or input box. Maximum frequency for this filter is 20.

The scatter chart visualizes the missed and made shots, while the tooltips display the distance and action type of the shots.
